## How It Works

**Step 1: Set up your environment (30 minutes)**
Follow the setup guide to install the prerequisites. If you've got Node.js already, this is mostly just installing the Expo CLI and connecting your phone.

**Step 2: Run the app locally (15 minutes)**
Clone the repo, install dependencies, and launch on your device using Expo Go. You'll see the app running with the sample dataset within minutes.

**Step 3: Make it yours (1-2 hours)**
Work through the customisation cookbook to adjust the app to your preferences. Change the colours, tweak the metrics, set your notification times. Each recipe is self-contained — pick the ones you care about.

**Step 4: Start tracking (ongoing)**
Log your first night's sleep. The app handles the rest: trend charts update automatically, weekly summaries generate every Monday, and your data stays on your device.

**Support:** You get access to a private GitHub Discussions board where you can ask questions, share customisations, and get help from other SleepSync users. Alex checks in weekly.
